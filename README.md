## Getting Started
Clone this repository
   
    git clone <insert-repository-url>

Change to the directory of the cloned repository then install the dependencies
    npm install

#### Running in Development
`gatsby develop`

#### Building
`gatsby build`

#### Deploying 
`gatsby deploy`
